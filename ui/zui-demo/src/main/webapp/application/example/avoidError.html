当前位置：易用性正例>十大原则>避免出错
避免出错：比良好的出错信息更好的办法是首先通过精心的设计来防止问题的发生。
正例列表  
序号 描述 上传日期
1 对于密码、帐号等重要信息的新增、修改，必须重复输入，避免产生严重后果。 2009-04-21
2 任务栏中，Windows应用程序右键菜单最后一项是“关闭”，而IM的是“关于”，经常会点错。应该把“关闭”放到最下面。 2009-04-21
3 在输入某些比较重要的信息的时候，不要等到用户输入完成以后再进行校验提示。建议鼠标移动到输入框控件上时就出现TIPS即时提示，防止用户出错。 2009-10-28
4 对于某些只需选当日之后的日期控件，当天之前的日期应为灰不可选择，防止用户在不经意的情况下选错日期。 2009-10-28
5 应当注意尽可能避免用户无意录入无效的数据，对可能引起致命错误或系统出错的输入字符或动作要加限制或屏蔽。 2009-11-2
